# 🌱 永劫种菜 · 游戏机制文档

> 本文档汇总了种菜系统的全部机制，包括基础作物数据、三类道具（各10级）的效果、消耗品（肥料 / 神偷手）的使用规则，以及收益计算逻辑。

---

## 一、基础设定

| 项目 | 数值 |
|------|------|
| 地块数量 | **6 块地** |
| 作物最低生长时间 | **15 分钟（0.25 小时）** — 任何缩短效果不能低于此值 |
| 道具分类 | **3 类**，每类最多选 1 个，同类内不可叠加 |
| 道具等级 | 每个道具 **1~10 级** |

### 异常状态

| 操作 | 效果 | 说明 |
|------|------|------|
| 💧 浇水 | 净减少生长时间 **1 小时** | 干旱状态下作物生长时间 +1h，浇水解除后净 −1h |
| 🪲 除虫 | 净增加产量 **+1** | 虫害状态下作物成熟后 −1，除虫后净 +1 |
| 🌿 除草 | 清除杂草 | 有杂草时地块无法种植和收获，清除后恢复正常；不影响时间/产量 |

> 💡 「清除异常」可触发净化沙漏和丰收骰子。
> 
> 💡 默认触发1次。可以修改。

## 三、第一类道具

### 3.1 🔔 速生钟 — 缩短生长时间

> **效果：** 当前作物生长时间缩短 **X 分钟**，每增加一级多减 15 分钟。

| 等级 | 缩短时间 | 折合小时 |
|:----:|:--------:|:--------:|
| Lv1 | 30 分钟 | 0.50 h |
| Lv2 | 45 分钟 | 0.75 h |
| Lv3 | 60 分钟 | 1.00 h |
| Lv4 | 75 分钟 | 1.25 h |
| Lv5 | 90 分钟 | 1.50 h |
| Lv6 | 105 分钟 | 1.75 h |
| Lv7 | 120 分钟 | 2.00 h |
| Lv8 | 135 分钟 | 2.25 h |
| Lv9 | 150 分钟 | 2.50 h |
| Lv10 | 165 分钟 | 2.75 h |

> **公式：** 缩短 = `(等级 + 1) × 15 分钟` = `(等级 + 1) × 0.25 小时`

---

### 3.2 🌿 共生苗 — 相邻作物产量加成

> **效果：** 与甘蔗相邻的其他作物增加 **X 点产量**。

| 等级 | 相邻作物每棵 +产量 |
|:----:|:------------------:|
| Lv1 | +1 |
| Lv2 | +2 |
| Lv3 | +3 |
| Lv4 | +4 |
| Lv5 | +5 |
| Lv6 | +6 |
| Lv7 | +7 |
| Lv8 | +8 |
| Lv9 | +9 |
| Lv10 | +10 |

> **公式：** 加成 = `等级 × 相邻甘蔗数`

**推荐混种布局（3🎋 + 3其他）— 6块地 3×2 网格：**

```
┌────┬────┐
│ 🎋 │ 🌱 │  ← 🌱 相邻 2 棵甘蔗
├────┼────┤
│ 🌱 │ 🎋 │  ← 🌱 相邻 3 棵甘蔗
├────┼────┤
│ 🎋 │ 🌱 │  ← 🌱 相邻 2 棵甘蔗
└────┴────┘
```

- 3 棵非甘蔗作物平均相邻甘蔗数：`(2 + 3 + 2) ÷ 3 ≈ 2.333` 棵
- 💡 **全甘蔗也有效：** 3×2 全甘蔗时，每棵甘蔗之间也相邻，4 个角各相邻 2 棵 + 2 条边各相邻 3 棵 → 平均 `(2+2+3+3+2+2) ÷ 6 ≈ 2.333` 棵

---

### 3.3 🎲 丰收骰子 — 概率触发，总产量乘法加成

> **效果：** 种植、收获、清除异常、被偷菜等行为有 **40% 概率** 增加 **X% 产量**，**仅生效 1 次**。
>
> 💡 **骰子是总产量乘法：** 触发时 `总产量 = 产量 × (1 + 骰子%)`，而不是只加基础产量。这意味着骰子对沙漏、除虫等所有加成都有放大效果。

由于「仅生效 1 次」是限制条件，实际触发率取决于触发事件的次数：

| 事件次数 | 场景说明 | P(≥1次触发) | 公式 |
|:--------:|:--------|:-----------:|:----:|
| 2次 | 仅种植+收获 | **64.0%** | 1−0.6² |
| 3次 | +1次清除异常/偷菜 | **78.4%** | 1−0.6³ |
| 4次 | +2次清除异常/偷菜 | **87.0%** | 1−0.6⁴ |
| 5次 | +3次清除异常/偷菜 | **92.2%** | 1−0.6⁵ |
| 6次 | +4次清除异常/偷菜 | **95.3%** | 1−0.6⁶ |
| 7次 | +5次清除异常/偷菜 | **97.2%** | 1−0.6⁷ |
| 8次 | +6次清除异常/偷菜 | **98.3%** | 1−0.6⁸ |
| 9次 | +7次清除异常/偷菜 | **99.0%** | 1−0.6⁹ |
| 10次 | +8次清除异常/偷菜 | **99.4%** | 1−0.6¹⁰ |

> 💡 作物生长时间越短、好友越少 → 异常/偷菜事件少 → 触发率偏低（接近 64%）。
> 💡 作物生长时间越长、好友越多 → 异常/偷菜事件多 → 可以按 **100% 必定触发**来算。

| 等级 | 骰子增幅 |
|:----:|:--------:|
| Lv1 | +5% |
| Lv2 | +10% |
| Lv3 | +15% |
| Lv4 | +20% |
| Lv5 | +25% |
| Lv6 | +30% |
| Lv7 | +35% |
| Lv8 | +40% |
| Lv9 | +45% |
| Lv10 | +50% |

> **公式：** `最终产量 = round(产量 × (1 + P(触发) × 5% × 等级))`
>
> 其中 P(触发) 根据事件次数计算，必定触发直接取 100%。骰子作用于**所有加成之后的总产量**（包括沙漏、除虫等），结果**四舍五入**取整。

## 四、第二类道具

### 4.1 🍶 醇化葫 — 成熟后额外成长

> **效果：** 作物成熟后额外成长 **2 小时**，可额外获得 **X 点产量**。

| 等级 | 额外产量 |
|:----:|:--------:|
| Lv1 | +2 |
| Lv2 | +3 |
| Lv3 | +4 |
| Lv4 | +5 |
| Lv5 | +6 |
| Lv6 | +7 |
| Lv7 | +8 |
| Lv8 | +9 |
| Lv9 | +10 |
| Lv10 | +11 |

> **公式：** 额外产量 = `等级 + 1`

> ⚠️ **注意：** 醇化葫的额外 2h 是作物成熟**之后**的增量，肥料不能缩短这 2 小时。因此带葫作物的**最短时长为 2.25h**（成熟 0.25h + 额外 2h）。

---

### 4.2 ⏳ 净化沙漏 — 清除异常获得额外产量

> **效果：** 每清除一次异常状态，可获得作物基础产量 **X%** 的额外产量。

| 等级 | 每次清除额外产量 |
|:----:|:---------------:|
| Lv1 | 3% |
| Lv2 | 4% |
| Lv3 | 6% |
| Lv4 | 7% |
| Lv5 | 9% |
| Lv6 | 10% |
| Lv7 | 12% |
| Lv8 | 13% |
| Lv9 | 15% |
| Lv10 | 16% |

> **公式：** 每块地额外产量 = `round(基础产量 × 对应等级%)`，总额外产量 = 每块地额外产量 × 清除次数
>
> ⚠️ **百分比计算取整规则：** 基础产量 × 百分比会出现小数，目前暂按**四舍五入（round）**取整，后续可能根据实测数据调整。

> ⚠️ **随机性较大**，可按 **1次 / 2次 / 3次** 三种触发频率来分类评估收益。

---

### 4.3 🎒 乾坤袋 — 偷菜额外收益

> **效果：** 每次偷菜可获得 **20%** 额外收益。

> 📌 **与种菜收益无关**，不影响种植利润计算。在收益计算器中作为占位项，等同于「无」。

## 六、消耗品

### 6.1 💩 肥料

| 属性 | 数值 |
|------|------|
| 价格 | **25 金币/个** |
| 效果 | 缩短目标作物生长时间 **1 小时** |
| 作用范围 | 1 个肥料只能作用 **1 块地** |
| 最短限制 | 作物最低缩短至 **15 分钟（0.25h）** |

> ⚠️ **与醇化葫的关系：** 醇化葫的 +2h 是成熟后的额外成长阶段，**肥料不能缩短这 2h**。带葫时最短总时长 = `0.25h（最短成熟）+ 2h（葫）= 2.25h`。

**计算逻辑：**

```
生长时间 = max(0.25, 基础时间 − 速生钟减少 − 肥料数 × 1)
总时长   = 生长时间 + (有醇化葫 ? 2 : 0)
净利润/h = (产量 × 单价 − 肥料成本 − 种子成本) ÷ 总时长
```

---

### 6.2 🥷 神偷手

| 属性 | 数值 |
|------|------|
| 价格 | **600 金币/次** |
| 效果 | 偷取目标农场所有可偷取作物的 **30% 收益** |
| 对方损失 | **无损失**（对方不受影响） |
| 偷后状态 | 对方作物立即进入 **保护状态** |

**计算逻辑：**

```
偷取总收益 = Σ(每块地产量 × 30% × 作物单价)
净赚      = 偷取总收益 − 600

净赚 > 0  → ✅ 值得出手
净赚 ≤ 0  → ❌ 亏本，别去
```

> 目标农场最多 **6 块地**，需逐块选择作物及产量进行评估。

## 九、核心收益公式汇总

```
利润/h = (产量 × 单价 − 种子成本) ÷ 生长时间

加成产量 = 基础产量
         + 共生苗加成（等级 × 相邻甘蔗数）        ← 仅混种/全甘蔗
         + 醇化葫加成（等级 + 1）
         + 净化沙漏加成（round(基础 × 等级%) × 清除次数）
         + 除虫加成（除虫次数 × 1）

最终产量 = round( 加成产量 × (1 + 骰子%) )          ← 骰子是总产量乘法！仅触发时生效

时间 = max(0.25, 基础时间 − 速生钟减少 − 肥料数 × 1 − 浇水次数 × 1)
     + 醇化葫额外（2h，有则加）

暴击加成 = 基础产量
         + 共生苗加成（等级 × 相邻甘蔗数）
         + 醇化葫加成（等级 + 1）
         + 净化沙漏加成（round(基础 × 等级%) × 清除次数）

暴击产量 = round( (暴击加成 × 2 + 除虫加成) × (1 + 骰子%) )  ← 骰子是总产量乘法

暴击利润/h = (暴击产量 × 单价 − 种子成本) ÷ 时间
```

> ⚠️ **暴击（产量翻倍）机制：** 作物有概率触发暴击，效果为 `(基础 + 沙漏 + 葫 + 共生苗) × 2 + 除虫`，即**生长阶段的加成（基础、沙漏、葫、共生苗）翻倍，除虫在翻倍后加，骰子最后乘**。已验证：辣椒基础 45 + 沙漏 Lv3 三次 = 54，暴击 ×2 = 108，除虫 +1 = 109 ✓
>
> ⚠️ **注意：** 暴击目前仅测试了沙漏的计算方式，共生苗和葫的加成是否也参与翻倍尚未实测验证，后续可能根据实测数据调整公式。
>
> ⚠️ **骰子作用在最终产量上：** `(基础 + 沙漏 + 除虫 + 其他) × (1 + 骰子%)`，不是只加基础。这也是为什么骰子配合沙漏/除虫收益特别高。
>
> ⚠️ **百分比取整规则：** 沙漏的 `基础产量 × 等级%` 和骰子的 `加成产量 × (1 + 骰子%)` 都会产生小数，目前暂按**四舍五入（round）**取整。后续可能根据实测数据调整。

> 💡 共生产苗模式下，6 块地按 **3🎋 + 3 其他**（或全🎋×6）混合计算，需考虑不同作物生长时间的最小公倍数周期。
