# 🌱 永劫种菜 · 游戏机制文档

> 本文档汇总了种菜系统的全部机制，包括基础作物数据、三类道具（各10级）的效果、消耗品（肥料 / 神偷手）的使用规则，以及收益计算逻辑。

---

## 一、基础设定

| 项目 | 数值 |
|------|------|
| 地块数量 | **6 块地** |
| 作物最低生长时间 | **15 分钟（0.25 小时）** — 任何缩短效果不能低于此值 |
| 道具分类 | **3 类**，每类最多选 1 个，同类内不可叠加 |
| 道具等级 | 每个道具 **1~10 级**（如意笔仅 **1~2 级**） |

### 异常状态

| 操作 | 效果 | 说明 |
|------|------|------|
| 💧 浇水 | 净减少生长时间 **1 小时** | 干旱状态下作物生长时间 +1h，浇水解除后净 −1h |
| 🪲 除虫 | 净增加产量 **+1** | 虫害状态下作物成熟后 −1，除虫后净 +1 |
| 🌿 除草 | 清除杂草 | 有杂草时地块无法种植和收获，清除后恢复正常；不影响时间/产量 |

> 💡 「清除异常」可触发净化沙漏和丰收骰子。
> 
> 💡 默认触发1次。可以修改。

### 3.1 🔔 速生钟 — 缩短生长时间

| 等级 | 缩短时间 | 折合小时 |
|:----:|:--------:|:--------:|
| Lv1 | 30 分钟 | 0.50 h |
| Lv2 | 45 分钟 | 0.75 h |
| Lv3 | 60 分钟 | 1.00 h |
| Lv4 | 75 分钟 | 1.25 h |
| Lv5 | 90 分钟 | 1.50 h |
| Lv6 | 105 分钟 | 1.75 h |
| Lv7 | 120 分钟 | 2.00 h |
| Lv8 | 135 分钟 | 2.25 h |
| Lv9 | 150 分钟 | 2.50 h |
| Lv10 | 165 分钟 | 2.75 h |

> **公式：** 缩短 = `(等级 + 1) × 15 分钟` = `(等级 + 1) × 0.25 小时`

### 3.2 🌿 共生苗 — 相邻作物产量加成

> **效果：** 与甘蔗相邻的其他作物增加 **X 点产量**。

| 等级 | 相邻作物 +产量 |
|:----:|:-------------:|
| Lv1 | +1 |
| Lv2 | +2 |
| Lv3 | +3 |
| Lv4 | +4 |
| Lv5 | +5 |
| Lv6 | +6 |
| Lv7 | +7 |
| Lv8 | +8 |
| Lv9 | +9 |
| Lv10 | +10 |

> ⚠️ **效果不可叠加：** 每棵作物只要相邻 ≥1 棵甘蔗即获得 +Lv 产量，相邻多棵甘蔗不会叠加。甘蔗与甘蔗相邻时，甘蔗自己也会获得加成。

**🏆 最优布局：2🎋 中排 + 4 其他（6/6 全覆盖）**

```
┌────┬────┐
│ 🌱 │ 🌱 │  ← 4 棵其他作物全部相邻甘蔗，获 +Lv
├────┼────┤
│ 🎋 │ 🎋 │  ← 2 棵甘蔗互相相邻，也获 +Lv
├────┼────┤
│ 🌱 │ 🌱 │  ← 全部 6 棵植物都获得加成！
└────┴────┘
```

- ✅ **仅需 2 棵甘蔗放在中排（C+D）**，即可让全部 6 块地获得加成
- ✅ 省下 4 块地种高利润作物（推荐萝卜，Lv4+ 开始反超辣椒）
- 📈 最优作物：Lv1~3 辣椒 → Lv4+ 萝卜（详细原因见 [共生苗详解.md](共生苗详解.md)）
- 🔗 **完整布局枚举、Lv1 验算步骤、Lv1~10 收益对比表** → 见 **[共生苗详解.md](共生苗详解.md)**

### 3.3 🎲 丰收骰子 — 概率触发，总产量乘法加成

> **效果：** 种植、收获、清除异常、被偷菜等行为有 **40% 概率** 增加 **X% 产量**，**仅生效 1 次**。
>
> 💡 **骰子是总产量乘法：** 触发时 `总产量 = 产量 × (1 + 骰子%)`，而不是只加基础产量。这意味着骰子对沙漏、除虫等所有加成都有放大效果。

由于「仅生效 1 次」是限制条件，实际触发率取决于触发事件的次数：

| 事件次数 | 场景说明 | P(≥1次触发) | 公式 |
|:--------:|:--------|:-----------:|:----:|
| 2次 | 仅种植+收获 | **64.0%** | 1−0.6² |
| 3次 | +1次清除异常/偷菜 | **78.4%** | 1−0.6³ |
| 4次 | +2次清除异常/偷菜 | **87.0%** | 1−0.6⁴ |
| 5次 | +3次清除异常/偷菜 | **92.2%** | 1−0.6⁵ |
| 6次 | +4次清除异常/偷菜 | **95.3%** | 1−0.6⁶ |
| 7次 | +5次清除异常/偷菜 | **97.2%** | 1−0.6⁷ |
| 8次 | +6次清除异常/偷菜 | **98.3%** | 1−0.6⁸ |
| 9次 | +7次清除异常/偷菜 | **99.0%** | 1−0.6⁹ |
| 10次 | +8次清除异常/偷菜 | **99.4%** | 1−0.6¹⁰ |

> 💡 作物生长时间越短、好友越少 → 异常/偷菜事件少 → 触发率偏低（接近 64%）。
> 💡 作物生长时间越长、好友越多 → 异常/偷菜事件多 → 可以按 **100% 必定触发**来算。

| 等级 | 骰子增幅 |
|:----:|:--------:|
| Lv1 | +5% |
| Lv2 | +10% |
| Lv3 | +15% |
| Lv4 | +20% |
| Lv5 | +25% |
| Lv6 | +30% |
| Lv7 | +35% |
| Lv8 | +40% |
| Lv9 | +45% |
| Lv10 | +50% |

> **公式：** `最终产量 = 向上取整(产量 × (1 + P(触发) × 5% × 等级))`
>
> 其中 P(触发) 根据事件次数计算，必定触发直接取 100%。骰子作用于**所有加成之后的总产量**（包括沙漏、除虫等），结果**向上取整**。

> **效果：** 作物成熟后额外成长 **2 小时**，可额外获得 **X 点产量**。

| 等级 | 额外产量 |
|:----:|:--------:|
| Lv1 | +2 |
| Lv2 | +3 |
| Lv3 | +4 |
| Lv4 | +5 |
| Lv5 | +6 |
| Lv6 | +7 |
| Lv7 | +8 |
| Lv8 | +9 |
| Lv9 | +10 |
| Lv10 | +11 |

> **公式：** 额外产量 = `等级 + 1`

> ⚠️ **注意：** 醇化葫的额外 2h 是作物成熟**之后**的增量，肥料不能缩短这 2 小时。因此带葫作物的**最短时长为 2.25h**（成熟 0.25h + 额外 2h）。

### 4.2 ⏳ 净化沙漏 — 清除异常获得额外产量

> **效果：** 每清除一次异常状态，可获得作物基础产量 **X%** 的额外产量。

| 等级 | 每次清除额外产量 |
|:----:|:---------------:|
| Lv1 | 3% |
| Lv2 | 4% |
| Lv3 | 6% |
| Lv4 | 7% |
| Lv5 | 9% |
| Lv6 | 10% |
| Lv7 | 12% |
| Lv8 | 13% |
| Lv9 | 15% |
| Lv10 | 16% |

> **公式：** 每块地额外产量 = `ceil(基础产量 × 对应等级%)`（向上取整），总额外产量 = 每块地额外产量 × 清除次数
>
> ⚠️ **取整规则：** 沙漏、骰子的产量计算均**向上取整**。
>
> ![](lv10钟+lv10沙漏+辣椒产量.png)
>
> 上图配置：🔔速生钟Lv10 + ⏳净化沙漏Lv10（16%）+ 除虫2次，种🌶️辣椒。
>
> ```
> 普通产量 63 = 45 + ceil(45×16%)×2 + 2 = 45 + 8×2 + 2 = 45 + 16 + 2
> 暴击产量 122 = 90 + ceil(90×16%)×2 + 2 = 90 + 15×2 + 2 = 90 + 30 + 2
> ```

> 💡 种植时间越长，触发频率可能越高（如甘蔗比小麦更容易触发），但即使是甘蔗也有可能1次都不触发，可自行估测次数。

## 六、消耗品

### 6.1 💩 肥料

| 属性 | 数值 |
|------|------|
| 价格 | **25 金币/个** |
| 效果 | 缩短目标作物生长时间 **1 小时** |
| 作用范围 | 1 个肥料只能作用 **1 块地** |
| 最短限制 | 作物最低缩短至 **15 分钟（0.25h）** |

> ⚠️ **与醇化葫的关系：** 醇化葫的 +2h 是成熟后的额外成长阶段，**肥料不能缩短这 2h**。带葫时最短总时长 = `0.25h（最短成熟）+ 2h（葫）= 2.25h`。

**计算逻辑：**

```
生长时间 = max(0.25, 基础时间 − 速生钟减少 − 肥料数 × 1)
总时长   = 生长时间 + (有醇化葫 ? 2 : 0)
净利润/h = (产量 × 单价 − 肥料成本 − 种子成本) ÷ 总时长
```

---

### 6.2 🥷 神偷手

| 属性 | 数值 |
|------|------|
| 价格 | **600 金币/次** |
| 效果 | 偷取目标农场所有可偷取作物的 **30% 收益** |
| 对方损失 | **无损失**（对方不受影响） |
| 偷后状态 | 对方作物立即进入 **保护状态** |

**计算逻辑：**

```
偷取总收益 = Σ(每块地产量 × 30% × 作物单价)
净赚      = 偷取总收益 − 600

净赚 > 0  → ✅ 值得出手
净赚 ≤ 0  → ❌ 亏本，别去
```

> 目标农场最多 **6 块地**，需逐块选择作物及产量进行评估。

## 九、核心收益公式汇总

```
利润/h = (产量 × 单价 − 种子成本) ÷ 生长时间

加成产量 = 基础产量
         + 共生苗加成（相邻甘蔗 ? 等级 : 0）        ← 不可叠加，仅混种/全甘蔗
         + 醇化葫加成（等级 + 1）
         + 净化沙漏加成（ceil(基础 × 等级%) × 清除次数）  ← 向上取整
         + 除虫加成（除虫次数 × 1）

最终产量 = 向上取整( 加成产量 × (1 + 骰子%) )      ← 骰子是总产量乘法！仅触发时生效

时间 = max(0.25, 基础时间 − 速生钟减少 − 肥料数 × 1 − 浇水次数 × 1)
     + 醇化葫额外（2h，有则加）

暴击：基础产量 × 2（种下瞬间触发，有特效）
     + 除虫加成（除虫次数 × 1）              ← 除虫不翻倍
     + 共生苗加成（相邻甘蔗 ? 等级 : 0）       ← 不翻倍
     + 醇化葫加成（等级 + 1）                 ← 不翻倍
     + 净化沙漏加成（ceil(基础产量×2 × 等级%) × 清除次数）  ← 在翻倍基数上重新ceil

暴击产量 = 向上取整( 以上之和 × (1 + 骰子%) )      ← 骰子是总产量乘法

暴击利润/h = (暴击产量 × 单价 − 种子成本) ÷ 时间
```

> ⚠️ **暴击机制：** 种下瞬间触发（有特效），**基础产量直接 ×2**。沙漏在翻倍后的基数上重新计算、除虫/共生苗/葫加在翻倍后、骰子最后乘（也向上取整）。已验证：辣椒45 + 沙漏Lv10(16%)×2次 + 除虫2 → 暴击产量122（=90 + 15×2 + 2）✓。暴击下共生苗、葫和骰子尚未实测验证，后续可能调整。
>
> ⚠️ **骰子作用在最终产量上：** `(基础 + 沙漏 + 除虫 + 其他) × (1 + 骰子%)`，不是只加基础。这也是为什么骰子配合沙漏/除虫收益特别高。
>
> 💡 **如意笔（🖌️）：** 装备后「基础产量」「基础时间」「单价」均替换为如意笔对应等级的值（见 §4.3），后续所有计算均在替换后的基础上进行。
>
> 💡 **产量取整：** 沙漏和骰子的产量计算均**向上取整**，确保**每块地的产量始终为整数**。
>
> 💡 共生苗模式下推荐 **2🎋 中排 + 4 其他**（`OOCCOO`），全部 6 块地均获得加成。每块地独立计算利润/h后相加即可。
