# 🌿 共生苗 · 完整机制与布局分析

> 本文档详细推导共生苗（Symbiotic Seedling）的全部游戏机制、最优布局策略和收益对比，供使用者理解并自行验算。

## 一、网格坐标系统

游戏共 **6 块地**，呈 **3 行 × 2 列** 排列。为便于讨论，使用字母 A~F 标注每块地：

```
┌────┬────┐
│ A  │ B  │   ← 第 1 行
├────┼────┤
│ C  │ D  │   ← 第 2 行（中排）
├────┼────┤
│ E  │ F  │   ← 第 3 行
└────┴────┘
```


## 二、共生苗效果规则

### 官方效果
> 与甘蔗相邻的其他作物增加 **X 点产量**，X = 共生苗等级。

### 实测验证的两条关键规则

| 规则 | 说明 | 实测验证 |
|:----|:-----|:--------|
| ✅ 不可叠加 | 无论相邻多少棵甘蔗，每棵作物只获得 **一次 +等级** 加成 | 全甘蔗时每棵只 +等级，不因相邻多棵而叠加 |
| ✅ 甘蔗自己也生效 | 甘蔗与甘蔗相邻时，甘蔗本身也获得 +等级 加成 | 全甘蔗 6 棵均获得加成 |

### 判定逻辑（伪代码）

```
对每块地 P:
  if P 是甘蔗:
    if P 相邻的地中至少有 1 棵甘蔗 → P 获得 +等级 产量
  else (P 是其他作物):
    if P 相邻的地中至少有 1 棵甘蔗 → P 获得 +等级 产量
```

### 计算公式

```
单块地产量 = 基础产量 + (相邻 ≥1 棵甘蔗 ? Lv : 0)

单块地利润/h = (产量 × 单价 - 成本) ÷ 生长时间

总利润/h = Σ 每块地利润/h
```

## 四、收益对比：a 棵甘蔗 + (6−a) 棵其他作物

从第三节的布局枚举可知：
- **a=1**：最多 3/6 加成（甘蔗放中排 C 或 D）
- **a≥2**：均可找到 6/6 全覆盖布局

因此只需比较这 6 种情况。每种情况取各自的最优布局，总利润/h 的通用公式为：

```
a=1:  甘蔗利润/h(无加成) + 3 × 作物利润/h(有加成) + 2 × 作物利润/h(无加成)
a≥2:  a × 甘蔗利润/h(有加成) + (6−a) × 作物利润/h(有加成)
```

其中：
- `甘蔗利润/h(无加成)` = (28 × 20 − 120) ÷ 22 = **20.00**
- `甘蔗利润/h(有加成)` = ((28+等级) × 20 − 120) ÷ 22
- `作物利润/h(无加成)` = (基础产量 × 单价 − 成本) ÷ 时间
- `作物利润/h(有加成)` = ((基础产量+等级) × 单价 − 成本) ÷ 时间

## 六、为什么萝卜是最优解？

### 核心原理

共生苗加的是**固定数值**（+等级），不是百分比。所以：

```
比例增益 = 等级 ÷ 基础产量 × 100%
```

| 等级 | 🌾 小麦 (基础15) | 🥬 白菜 (基础25) | 🥕 萝卜 (基础15) | 🌶️ 辣椒 (基础45) |
|:--:|:---------------:|:-----------------:|:----------------:|:-----------------:|
| 1 | +6.7% | +4.0% | **+6.7%** | +2.2% |
| 3 | +20.0% | +12.0% | **+20.0%** | +6.7% |
| 5 | +33.3% | +20.0% | **+33.3%** | +11.1% |
| 7 | +46.7% | +28.0% | **+46.7%** | +15.6% |
| 10 | **+66.7%** | +40.0% | **+66.7%** | +22.2% |

> 萝卜和小麦的基础产量相同（都是 15），但萝卜单价 17、小麦仅 5——所以同样的 +等级 加成，萝卜的利润提升是小麦的 3.4 倍！

### 萝卜 vs 辣椒 临界点推算

```
辣椒 等级1 利润/h：25.42（无加成）→ 26.08（有加成） 增量 +0.67
萝卜 等级1 利润/h：22.00（无加成）→ 23.70（有加成） 增量 +1.70

2🎋+4辣椒 等级1 = 20.91×2 + 26.08×4 = 41.82 + 104.33 = 146.15
2🎋+4萝卜 等级1 = 20.91×2 + 23.70×4 = 41.82 + 94.80  = 136.62
差距 = 9.53 ← 辣椒领先

但随着等级升高：
萝卜每级增量 = 4 × (17÷10) = 4 × 1.70 = 6.80/h
辣椒每级增量 = 4 × (8÷12)  = 4 × 0.67 = 2.67/h

萝卜每升一级追回 6.80 − 2.67 = 4.13

等级1 差距 9.53，等级2 差距 5.40，等级3 差距 1.27...
等级4：萝卜反超 162.47 vs 159.61 = +2.86 ✓
```

> 📐 **临界点：等级4。** 等级≥4 后萝卜始终最优，且优势随等级不断扩大。
