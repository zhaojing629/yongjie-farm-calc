# 🌱 永劫种菜 · 游戏机制文档

> 本文档汇总了种菜系统的全部机制，包括基础作物数据、三类道具（各10级）的效果、消耗品（肥料 / 神偷手）的使用规则，以及收益计算逻辑。

---

## 一、基础设定

| 项目 | 数值 |
|------|------|
| 地块数量 | **6 块地** |
| 作物最低生长时间 | **15 分钟（0.25 小时）** — 任何缩短效果不能低于此值 |
| 道具分类 | **3 类**，每类最多选 1 个，同类内不可叠加 |
| 道具等级 | 每个道具 **1~10 级** |

## 三、第一类道具

### 3.1 🔔 速生钟 — 缩短生长时间

> **效果：** 当前作物生长时间缩短 **X 分钟**，每增加一级多减 15 分钟。

| 等级 | 缩短时间 | 折合小时 |
|:----:|:--------:|:--------:|
| Lv1 | 30 分钟 | 0.50 h |
| Lv2 | 45 分钟 | 0.75 h |
| Lv3 | 60 分钟 | 1.00 h |
| Lv4 | 75 分钟 | 1.25 h |
| Lv5 | 90 分钟 | 1.50 h |
| Lv6 | 105 分钟 | 1.75 h |
| Lv7 | 120 分钟 | 2.00 h |
| Lv8 | 135 分钟 | 2.25 h |
| Lv9 | 150 分钟 | 2.50 h |
| Lv10 | 165 分钟 | 2.75 h |

> **公式：** 缩短 = `(等级 + 1) × 15 分钟` = `(等级 + 1) × 0.25 小时`

---

### 3.2 🌿 共生苗 — 相邻作物产量加成

> **效果：** 与甘蔗相邻的其他作物增加 **X 点产量**。

| 等级 | 相邻作物每棵 +产量 |
|:----:|:------------------:|
| Lv1 | +1 |
| Lv2 | +2 |
| Lv3 | +3 |
| Lv4 | +4 |
| Lv5 | +5 |
| Lv6 | +6 |
| Lv7 | +7 |
| Lv8 | +8 |
| Lv9 | +9 |
| Lv10 | +10 |

> **公式：** 加成 = `等级 × 相邻甘蔗数`

**推荐混种布局（3🎋 + 3其他）— 6块地 3×2 网格：**

```
┌────┬────┐
│ 🎋 │ 🌱 │  ← 🌱 相邻 2 棵甘蔗
├────┼────┤
│ 🌱 │ 🎋 │  ← 🌱 相邻 3 棵甘蔗
├────┼────┤
│ 🎋 │ 🌱 │  ← 🌱 相邻 2 棵甘蔗
└────┴────┘
```

- 3 棵非甘蔗作物平均相邻甘蔗数：`(2 + 3 + 2) ÷ 3 ≈ 2.333` 棵
- 3 棵甘蔗之间**无**共生苗加成（因为只有甘蔗才算"相邻"来源）
- ⚠️ **全甘蔗也有效：** 3×2 全甘蔗时，每棵甘蔗之间也相邻，4 个角各相邻 2 棵 + 2 条边各相邻 3 棵 → 平均 `(2+2+3+3+2+2) ÷ 6 ≈ 2.333` 棵

---

### 3.3 🎲 丰收骰子 — 概率触发产量加成

> **效果：** 种植、收获、清除异常、被偷菜等行为有 **40% 概率** 增加 **X% 作物产量**，**仅生效 1 次**。

由于「仅生效 1 次」是限制条件，实际触发率取决于触发事件的次数。计算器提供 5 种选项（切换到骰子时默认选「必定触发」）：

| 事件次数 | 场景说明 | P(≥1次触发) | 公式 |
|:--------:|:--------|:-----------:|:----:|
| 2次 | 仅种植+收获 | **64%** | 1−0.6² |
| 3次 | +1次清除异常/偷菜 | **78%** | 1−0.6³ |
| 4次 | +2次清除异常/偷菜 | **87%** | 1−0.6⁴ |
| 5次 | +3次清除异常/偷菜 | **92%** | 1−0.6⁵ |
| 必定触发 | 高频事件 → 等价100% | **100%**（默认） | — |

| 等级 | 2次(64%) | 3次(78%) | 4次(87%) | 5次(92%) | 必定(100%) |
|:----:|:--------:|:--------:|:--------:|:--------:|:----------:|
| Lv1 | +3.2% | +3.9% | +4.4% | +4.6% | +5% |
| Lv2 | +6.4% | +7.8% | +8.7% | +9.2% | +10% |
| Lv3 | +9.6% | +11.8% | +13.1% | +13.8% | +15% |
| Lv4 | +12.8% | +15.7% | +17.4% | +18.4% | +20% |
| Lv5 | +16.0% | +19.6% | +21.8% | +23.1% | +25% |
| Lv6 | +19.2% | +23.5% | +26.1% | +27.7% | +30% |
| Lv7 | +22.4% | +27.4% | +30.5% | +32.3% | +35% |
| Lv8 | +25.6% | +31.4% | +34.8% | +36.9% | +40% |
| Lv9 | +28.8% | +35.3% | +39.2% | +41.5% | +45% |
| Lv10 | +32.0% | +39.2% | +43.5% | +46.1% | +50% |

> **公式：** 产量加成 = `基础产量 × P(触发) × 5% × 等级`，其中 P(触发) = `1 − 0.6^事件次数`，必定触发直接取 100%。

## 四、第二类道具

### 4.1 🍶 醇化葫 — 成熟后额外成长

> **效果：** 作物成熟后额外成长 **2 小时**，可额外获得 **X 点产量**。

| 等级 | 额外产量 |
|:----:|:--------:|
| Lv1 | +2 |
| Lv2 | +3 |
| Lv3 | +4 |
| Lv4 | +5 |
| Lv5 | +6 |
| Lv6 | +7 |
| Lv7 | +8 |
| Lv8 | +9 |
| Lv9 | +10 |
| Lv10 | +11 |

> **公式：** 额外产量 = `等级 + 1`

> ⚠️ **注意：** 醇化葫的额外 2h 是作物成熟**之后**的增量，肥料不能缩短这 2 小时。因此带葫作物的**最短时长为 2.25h**（成熟 0.25h + 额外 2h）。

---

### 4.2 ⏳ 净化沙漏 — 清除异常获得额外产量

> **效果：** 每清除一次异常状态，可获得作物基础产量 **X%** 的额外产量。

| 等级 | 每次清除额外产量 |
|:----:|:---------------:|
| Lv1 | 3% |
| Lv2 | 4% |
| Lv3 | 6% |
| Lv4 | 7% |
| Lv5 | 9% |
| Lv6 | 10% |
| Lv7 | 12% |
| Lv8 | 13% |
| Lv9 | 15% |
| Lv10 | 16% |

> **公式：** 总额外产量 = `基础产量 × 对应等级% × 清除次数`

> ⚠️ **随机性较大**，可按 **1次 / 2次 / 3次** 三种触发频率来分类评估收益。

---

### 4.3 🎒 乾坤袋 — 偷菜额外收益

> **效果：** 每次偷菜可获得 **20%** 额外收益。

> 📌 **与种菜收益无关**，不影响种植利润计算。在收益计算器中作为占位项，等同于「无」。

> **效果：** 他人偷菜被抓概率为 **X%**（以 Lv3 为例：9%）。

> 📌 **与种菜收益无关**，仅影响防御偷菜。在收益计算中不作考虑。

## 六、消耗品

### 6.1 💩 肥料

| 属性 | 数值 |
|------|------|
| 价格 | **25 金币/个** |
| 效果 | 缩短目标作物生长时间 **1 小时** |
| 作用范围 | 1 个肥料只能作用 **1 块地** |
| 最短限制 | 作物最低缩短至 **15 分钟（0.25h）** |

> ⚠️ **与醇化葫的关系：** 醇化葫的 +2h 是成熟后的额外成长阶段，**肥料不能缩短这 2h**。带葫时最短总时长 = `0.25h（最短成熟）+ 2h（葫）= 2.25h`。

**计算逻辑：**

```
生长时间 = max(0.25, 基础时间 − 速生钟减少 − 肥料数 × 1)
总时长   = 生长时间 + (有醇化葫 ? 2 : 0)
净利润/h = (产量 × 单价 − 肥料成本 − 种子成本) ÷ 总时长
```

---

### 6.2 🥷 神偷手

| 属性 | 数值 |
|------|------|
| 价格 | **600 金币/次** |
| 效果 | 偷取目标农场所有可偷取作物的 **30% 收益** |
| 对方损失 | **无损失**（对方不受影响） |
| 偷后状态 | 对方作物立即进入 **保护状态** |

**计算逻辑：**

```
偷取总收益 = Σ(每块地产量 × 30% × 作物单价)
净赚      = 偷取总收益 − 600

净赚 > 0  → ✅ 值得出手
净赚 ≤ 0  → ❌ 亏本，别去
```

> 目标农场最多 **6 块地**，需逐块选择作物及产量进行评估。

## 九、核心收益公式汇总

```
利润/h = (产量 × 单价 − 种子成本) ÷ 生长时间

产量 = 基础产量
     + 丰收骰子加成（基础 × P(触发) × 5% × 等级）  ← P=1−0.6^N，默认必定=100%
     + 共生苗加成（等级 × 相邻甘蔗数）        ← 仅混种/全甘蔗
     + 醇化葫加成（等级 + 1）
     + 净化沙漏加成（基础 × 等级% × 清除次数）

时间 = max(0.25, 基础时间 − 速生钟减少 − 肥料数 × 1)
     + 醇化葫额外（2h，有则加）
```

> 共生产苗模式下，6 块地按 **3🎋 + 3 其他**（或全🎋×6）混合计算，需考虑不同作物生长时间的最小公倍数周期。
